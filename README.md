# Management:
```text
Movement -- w-s-d-a
Select the -- m spell and the spell number on the right
Some spells require a trigger location, to do this, left-click on the desired cell.
Switch to ranged combat -- r
Shoot the arrows
Melee is initially enabled, you just have to go to the opponent's cage.
Save the game -- F5, load the game -- F9
Log out of the game -- ESC

At the end of each level, you can upgrade your hero.
```

# Launch the game: 
```bash
make && ./game
```
